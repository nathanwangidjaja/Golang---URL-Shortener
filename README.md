# Golang---URL-Shortener

In the Terminal: 
```
cd [Directory for clone]
```
Run the main.go 

```
go run main.go
```

Go to localhost:8086 

Directories include:
/go = https://golang.org
/git = https://github.com/nathanwangidjaja

Therefore, when going to localhost:8086/go , the path will redirect you to the golang web server
